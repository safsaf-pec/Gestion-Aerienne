# Gestion Aérienne (Projet en C)
Projet en langage C developpé avec Code::Blocks.
## Description
Ce projet permet de gérer une application simple en C (Structure, fonctions , menu interactif, etc...).
Il a été réalisé dans le cadre d'un cours de programmation en langage C .
## Technologies
- Langage C
- Code::Blocks
- GCC
## Compilation et exécution 
1. Ouvrir le projet avec Code::Blocks
2. Compiler le projet (Build)
3. Exécuter le programme (Run)
